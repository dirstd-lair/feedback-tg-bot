# 🤖 Site Feedback Telegram Bot

Асинхронный Telegram-бот для сбора обратной связи и сообщений об ошибках. Построен на Python 3.12, Aiogram 3 и Docker.

---

## ⚙️ Конфигурация проекта

### 1. Переменные окружения (.env)
Создайте файл `.env` в корне проекта:


| Ключ | Описание |
| :--- | :--- |
| **TOKEN** | API-токен бота от @BotFather |

### 2. Внутренние настройки (в коде)
* **База данных**: SQLite (`aiosqlite`). Путь: `database/database.db`
* **Администрирование**:
    * `ADMIN_ID_CHAT`: `-5279863189` (ID группы для заявок)
    * `ADMIN_USERNAME`: `@username` (менеджер для связи)

---

## 🕒 Сценарий диалога
1. **Старт**: Приветствие пользователя.
2. **Причина**: Ввод краткой темы обращения.
3. **Подробности**: Ввод детального описания проблемы.
4. **Финал**: Отправка отчета в админ-чат.
